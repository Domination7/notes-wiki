Conics

A Conics are a curve gotten from cutting a right circular cone with a plane. The four conics are circles, ellipses, parabolas, and hyperbolas. Each have a different standard and general form. The formula for a conic is ax2 + bx + cy2 + dy = 1, if a and c are equal, the conic is a circle. If a times c is negative, the conic is a hyperbola. If there is no a or c, the conic is a prabola. If a times c is positive, the conic is an ellipse.

Questions

What is the shape given by 2x2+2x+2y2+3y=36?

The answer is a circle.
